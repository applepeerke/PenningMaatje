PenningMaatje
----------------------------------------------------------------------------------------
Author : Peter Heijligers
Version: 1.0
Python : >= 3.8
----------------------------------------------------------------------------------------
usage:
1. cd to the PenningMaatje directory.
2. python pm.py

Output
------
All output is directed to PenningMaatje subfolder Output (by default).
Also, a logfile is created.


Parameters (optional)
---------------------
To see the CLI input parameters, run "python pm -h".

-i = input directory
Bank transactions directory. This folder must contain bank transaction csv files only.

-o = output directory
Output directory for the database, log, backup, exports.

-y = year to export (optional).
Default is the current year.
